Cursor Prompt · Dùng cái này để code agent build folder feature
## Prompt cho Cursor/Claude Code — Paste vào chat Build a Chrome extension content script that injects a folder sidebar into chatgpt.com. Requirements: 1. Inject a collapsible sidebar on the LEFT of the existing ChatGPT sidebar 2. React component using Shadow DOM (to avoid CSS conflicts with ChatGPT) 3. Features needed: - Create/rename/delete folders with color picker - Drag and drop chats from ChatGPT sidebar into folders - MutationObserver to watch for new chats being added - Persist folder structure to chrome.storage.sync - Collapse/expand folders 4. Use TypeScript, Tailwind CSS inside Shadow DOM 5. Handle SPA navigation (ChatGPT uses React Router — sidebar re-renders) 6. Must work with Manifest V3 (no eval, no remote code) The ChatGPT sidebar chat list items have this structure: [inspect chatgpt.com and paste the actual DOM structure here] Start with the FolderTree component and the storage utilities.
⚠️ Trick quan trọng nhất: Dùng Shadow DOM để inject UI — CSS của extension không bị conflict với CSS của ChatGPT/Claude. Đây là cách Superpower ChatGPT làm. Nếu không dùng Shadow DOM, UI sẽ bị vỡ khi ChatGPT update CSS của họ.

Rủi ro & Cách xử lý

🔄
ChatGPT thay đổi DOM — Extension bị break
Rủi ro #1. OpenAI update UI thường xuyên, DOM selectors thay đổi. Superpower ChatGPT đã bị break nhiều lần. Users sẽ leave 1-star review ngay lập tức.
→ Fix: Abstract tất cả selectors vào 1 file config. Setup Playwright monitor chạy hàng ngày — alert khi DOM thay đổi. Commit để fix trong vòng 24h. Đây là competitive moat: respond nhanh hơn Superpower.
🏢
OpenAI native-build folders/search
Nếu OpenAI build tính năng này vào ChatGPT Plus, extension mất giá trị với ChatGPT users. Xác suất thấp trong 12 tháng — họ focus vào model AI, không UX.
→ Fix: Multi-platform từ tháng 1 (Claude.ai). Nếu ChatGPT build folders, anh vẫn có Claude + Gemini market. Đây là lý do critical phải support multi-platform sớm.
📜
Chrome Web Store reject extension
Google review có thể reject vì broad host permissions. Review process mất 1–3 tuần, có thể bị delay launch.
→ Fix: Chỉ request permissions cần thiết. Viết privacy policy rõ ràng ("data stored locally only"). Submit sớm 2 tuần trước launch date. Nếu bị reject, fix trong 24h và resubmit.
💸
Conversion rate thấp hơn 3%
Nếu chỉ 1% users convert, revenue giảm 2/3. Easy Folders ở 1.25%, Superpower ở 10%. Khoảng này phụ thuộc rất nhiều vào product quality và timing của upsell.
→ Fix: A/B test upgrade prompt timing. Show upsell sau khi user có action value rõ ràng (e.g., sau khi tạo folder thứ 3). Email sequence ngay khi free limit hit. Lifetime deal giúp bypass subscription friction.
